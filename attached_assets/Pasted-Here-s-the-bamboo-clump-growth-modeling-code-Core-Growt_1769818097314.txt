Here's the bamboo clump growth modeling code:

Core Growth Model: BambooClump Class
The simulation uses a logistic (S-curve) growth model - the same pattern seen in real plant growth:

Height Calculation (Lines 304-323)
private calculateHeight(progress: number): number {
  const maxHeight = 2.55;   // Maximum visual height (scaled for 3D scene)
  const minHeight = 0.5;    // Start with small saplings
  const growthRate = 15;    // Steepness - how fast growth accelerates
  const midpoint = 0.3;     // Peak growth year (year 3 = 2028)
  
  // Logistic growth formula (S-curve)
  return minHeight + (maxHeight - minHeight) / 
         (1 + Math.exp(-growthRate * (progress - midpoint)));
}
How it works:

progress ranges from 0 (year 2025) to 1 (year 2035)
Growth is slow at first, accelerates rapidly around year 3, then plateaus
This mimics real bamboo which has explosive growth in early maturity
Pole Count (Lines 325-340)
private calculatePoleCount(progress: number): number {
  const minPoles = 5;
  const maxPoles = 100;
  
  // Same S-curve for natural clump density
  return minPoles + (maxPoles - minPoles) / 
         (1 + Math.exp(-15 * (progress - 0.3)));
}
Carbon Sequestration (Lines 342-346)
private calculateCarbon(height: number, poleCount: number): number {
  // Based on Mindanao research: 87.5 tonnes CO2/ha/year at full maturity
  return (height / 27) * (poleCount / 90) * 87.5;
}

The S-curve (logistic function) creates realistic growth where:

Years 1-2: Slow establishment phase
Years 2-4: Explosive growth period (midpoint at year 3)
Years 5+: Growth plateaus as bamboo reaches maturity
